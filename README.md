#  Генератор Эвентов

Веб-приложение для управления эвентами и раздачи призов с функцией генерации случайных чисел и автоматическим запоминанием участников.

##  Основные возможности

###  Генерация случайных чисел
- Ввод списка участников через запятую
- Генерация определенного количества уникальных случайных чисел
- Автоматическое запоминание соответствия "номер  участник"
- Отображение сгенерированных чисел с возможностью копирования

###  Выдача призов
- Выбор типа награды: маты, деньги, коины
- Ввод точного количества награды
- Автоматическое заполнение статиков из выбранных участников
- Генерация команд в формате:
  - `!smatsa [статистики] количество` (для матов)
  - `!scasha [статистики] количество` (для денег)
  - `!coinsa [статистики] количество` (для коинов)

###  Система запоминания
- Автоматическое сохранение выбранных участников после генерации чисел
- Автоматическое заполнение статиков при переходе в раздел "Выдача призов"
- Кнопка для ручного заполнения статиков из выбранных участников
- Визуальные уведомления о статусе операций

##  Дизайн

##  Как использовать

### 1. Регистрация участников
1. Введите список участников через запятую без пробелов
2. Нажмите "Зарегистрировать участников"
3. Система покажет зарегистрированных участников

### 2. Генерация случайных чисел
1. Выберите "Генерация случайных чисел"
2. Введите количество чисел для генерации
3. Нажмите "Сгенерировать числа"
4. Система покажет числа и соответствие "номер  участник"
5. **Статики автоматически сохраняются** для использования в командах

### 3. Выдача призов
1. Выберите "Выдача призов"
2. Выберите тип награды (маты/деньги/коины)
3. Введите количество награды
4. **Статики автоматически заполняются** из выбранных участников
5. Или используйте кнопку " Заполнить из выбранных участников"
6. Нажмите "Сгенерировать команду"
7. Скопируйте готовую команду

##  Пример использования

1. **Участники:** `3,4,70,9,1`
2. **Генерируете 3 числа:** получаете `2,4,5`
3. **Система запоминает:** статики `4,9,1`
4. **В команде автоматически:** `!smatsa [4,9,1] 100`
